Dashboard


Dans le cadre de Notre Projet Data Science, nous avons choisi de visualiser l'ensemble de données sur les incidents criminels dans la ville de Los Angeles entre 2010 et 2015 . Différentes techniques de visualisations de données existent. Pour notre études nous avons utilisé 3 techniques:
  • Map
  • Histogramme
  • Parallel coordinate
Les données sont transcrites à partir des rapports de criminels originaux sur papiers. Il est donc possible que certains données soient inexactes. Les champs d’adresse ne sont fournis qu'à titre indicatif pour préserver la confidentialité.
